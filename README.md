# HisaabBook 

A personal finance tracker app built with Flutter & SQFlite.

## Features
- User Login & Sign Up UI screens
- Local database storage using SQFlit
-  View transactions
- Add Income & Expense transactions
- Delete transactions with long press

## Widgets Used
- Form & Validation
- Bottom Navigation Bar
- ListView.builder
- ElevatedButton
- Checkbox, Radio Button
- TextFormField
- RefreshIndicator
- FloatingActionButton
- AppBar
- DropdownButton
- Switch
- ListTile
- Drawer

## OOP Concepts Used
- Classes and Objects
- Encapsulation
- Inheritance
- Abstract Classes

## Tech Stack
- Flutter & Dart
- SQFlite (Local Database)
- Android Studio
